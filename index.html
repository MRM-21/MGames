Games

Both this games were made using HTML, CSS and JAVA code.

For the first game (Tic-Tac-Toe), I watched several tutorials about JAVA coding, were I was able to learn functions like: "if","return","const","let","math","loop", among many others. This videos were very important for the development of the game: video 1 , video 2 , video 3 and video 4 . As for the second game, (Snake) it was quite challenging, because I had to create a canvas (the box that contains the snake) at first I was not able to make the snake stay inside of it, until I found this video explaning all the steps on how to code the snake game. I also had to undestand more functions, such as: "getRandomInt", "log", "max and min", this Youtube video helped me understand most of them. It took me almost 5 days to complete the first game (Tic-Tac-Toe) and a week and a half to make the second one (Snake). I hope you like them both. Enjoy!
